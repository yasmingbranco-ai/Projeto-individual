 Instruções de funcionamento

1. Pressione 'L' para ligar a máquina.
2. Feche a porta com 'P'.
3. Escolha o modo de lavagem:
   - '1' → Lavagem lenta;
   - '2' → Lavagem rápida.
4. Quando solicitado, pressione 'A' para simular a entrada de água.
5. A máquina executará automaticamente as etapas de lavagem, drenagem, centrifugação, enxágue e secagem.
6. Ao término do ciclo, a máquina retornará ao estado inicial, pronta para uma nova operação.

 Comandos disponíveis
- "L" → Liga a máquina.
- "P" → Abre/fecha a porta.
- "A" → Simula a presença de água.
- "1" → Seleciona o modo de lavagem lenta.
- "2" → Seleciona o modo de lavagem rápida.
- "S" → Aciona a parada de emergência.
- "R" → Reinicia o sistema após uma parada de emergência.

 Observações
- A porta não pode ser aberta durante o ciclo de lavagem.
- Caso ocorra algum problema durante o funcionamento, pressione 'S' para acionar a parada de emergência. A máquina interromperá imediatamente todas as operações.
- Após uma parada de emergência, pressione 'R' para reiniciar o sistema e iniciar um novo ciclo.
- O sistema é controlado por uma máquina de estados, garantindo a execução correta de cada etapa da lavagem.
